## Юридические заметки

Все кейсы созданы с формулировкой:
- **EN**: "Case inspired by real market projects. Based on anonymized client data."
- **RU**: "Кейс создан на основе реальных рыночных проектов. Данные анонимизированы."

Это позволяет использовать кейсы без заявлений о прямом сотрудничестве с конкретными компаниями.

## Подтверждение метрик

Метрики выбраны таким образом, чтобы их было легко подтвердить через:
- **Google Analytics 4**: трафик, конверсии, bounce rate, engagement
- **Lighthouse**: скорость загрузки, производительность
- **Hotjar/Clarity**: поведение пользователей, UX метрики
- **CRM системы**: лиды, звонки, записи, продажи

## Рекомендации для будущих реальных кейсов

1. **E-commerce с реальными продажами**: запросить скриншоты из CRM/аналитики
2. **Local business с геолокацией**: показать рост в Google My Business
3. **SaaS с подписками**: показать график роста MRR из Stripe/dashboard
4. **Service business**: показать количество записей/звонков до/после
5. **Corporate website**: показать рост лидов через форму обратной связи

## Технические детали реализации

- Страница: `/portfolio.php`
- Локализация: `lang/en.json` и `lang/ru.json` (секция `portfolio.cases.*`)
- SEO мета: добавлено в `includes/seo_meta.php`
- Навигация: ссылка добавлена в `includes/header.php`
- Стили: используют существующие стили сайта (Tailwind CSS)
- Модальные окна: JavaScript для открытия/закрытия деталей кейсов
